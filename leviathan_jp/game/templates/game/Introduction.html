実験へようこそ

この実験は全{{ num_rounds }}ラウンドで構成されています。 あなたは{{ group_size }}人のグループに属しています。

ゲームの流れ

  1. 各ラウンドの開始時に、各プレイヤーは{{ endowment }}実験円を受け取ります
  2. この{{ endowment }}実験円を「個人口座」と「公共口座」に配分してください
  3. 公共口座に入れた金額の合計は{{ efficiency_factor }}倍され、グループメンバー全員に均等に分配されます
  4. 個人口座に残した金額はそのままあなたの収益となります

収益の計算

各ラウンドのあなたの収益は以下のように計算されます：

収益 = 個人口座の金額 + (公共口座の総額 × {{ efficiency_factor }} ÷ {{ group_size }})

懲罰メカニズム

各ラウンドで貢献額を決定した後、他のプレイヤーに対して懲罰を与えることができます。

  • 各プレイヤーは初期に{{ initial_punishment_endowment }}ポイントの懲罰権を持っています
  • 1ポイントの懲罰権を使用すると、相手に{{ punishment_rate }}実験円の損失を与えます
  • 懲罰を与えるには1ポイントの懲罰権が必要です（コスト：1実験円）
{{ if treatment == 'transfer_free' or treatment == 'transfer_cost' }}

懲罰権の譲渡

この実験では、懲罰権を他のプレイヤーに譲渡することができます。

  • 懲罰権は{{ punishment_transfer_unit }}ポイント単位で譲渡可能です
    • {{ if treatment == 'transfer_cost' }}
  • 譲渡にはコストがかかります：{{ punishment_transfer_unit }}ポイントあたり{{ punishment_transfer_cost_rate }}実験円
    • {{ else }}
  • 譲渡にコストはかかりません
    • {{ endif }}
  • 譲渡された懲罰権は次のラウンドから使用可能です
{{ endif }}

重要な注意事項

  • すべての決定は同時に行われます
  • 他のプレイヤーの決定を見る前に、あなたの決定を行ってください
  • 各ラウンドの結果は、全員の決定後に表示されます
    • {{ if treatment == 'fixed' }}
  • 懲罰権の保有量は全ラウンドを通じて一定です
    • {{ else }}
  • 使用した懲罰権は減少し、譲渡された懲罰権は増加します
    • {{ endif }}
